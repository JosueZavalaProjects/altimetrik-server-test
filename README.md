## Getting Started

First, install dependencies:

```bash
npm i
# or
npm install
# or
yarn
# or
yarn i

```

Later, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5001/books](http://localhost:5001/books) with your browser to see the result.
